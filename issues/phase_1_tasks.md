# Phase 1: スクレイピング＋PostgreSQLセットアップ

## 親タスク

1. **Supabaseプロジェクト作成**  
   #### 子タスク  
   - 管理コンソールでプロジェクトを作成。
   - APIキーを取得。
   - 接続URLを環境変数に設定。

2. **テーブル設計＋Row Level Security (RLS)設定**  
   #### 子タスク  
   - 必要なフィールドを設計。
   - Row Level Securityに基づくアクセス制御を適用。

3. **GitHub Actionsでスクレイパー実装**  
   #### 子タスク  
   - ワークフローの作成。
   - APIを用いた権限セット付与のテスト。
